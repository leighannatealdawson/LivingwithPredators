# Species photos

This folder holds the two identification photos shown on the Intro page of the
survey, plus an optional consent image.

| File | Referenced from | Status |
|------|-----------------|--------|
| `fox.jpg` | Intro page, question `species_f` ("Photo A") | **placeholder** — replace with a licensed fox photo before launch |
| `pm.jpg` | Intro page, question `species_pm` ("Photo B") | **placeholder** — replace with a licensed pine marten photo before launch |
| `Consent.png` | Welcome page (top of consent notice) | optional — drop a file with this name if you want a header image, otherwise the welcome page renders text-only |

## Replacing the placeholders

1. Drop the replacement files in this folder with the exact filenames above.
2. Commit and push — the deploy workflow will pick them up automatically.

**Licensing reminder:** the placeholders currently in this folder come from
[loremflickr](https://loremflickr.com) (themed Flickr results) and are for
development-time preview only. The researcher must supply photos whose
licence permits use in academic research before the survey goes public.
